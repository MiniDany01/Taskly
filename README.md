# Taskly

Taskly is a SaaS web application for managing school tasks, subjects, and reminders.

## Features

- User authentication with JWT
- Subject management with custom colors
- Task management (pending / completed)
- Urgent tasks system
- Dashboard with statistics
- Calendar view
- Task reminders

## Tech Stack

Frontend:

- HTML
- CSS
- JavaScript

Backend:

- Node.js
- Express

Database:

- PostgreSQL
- Prisma ORM

## Installation

Clone the repository

git clone https://github.com/MiniDany01/taskly.git

Install dependencies

npm install

Run the server

npm run dev
